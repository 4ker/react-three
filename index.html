React-Three

An example of using the React library to control a 3D scene using three.js

React-three adds additional elemental components to the current React library. These components represent the basic primitives found in three.js such as Mesh, Scene, Object3D.

To create a 3D scene you can create and compose React-three components just as you would compose DOM elements in vanilla React. An example:

var Cupcake = React.createClass({
  displayName: 'Cupcake',
  render: function() {
    return ReactTHREE.Object3D(
      {rotation:this.props.rotation, position:new THREE.Vector3(0,0,0)},
      ReactTHREE.Mesh({position:new THREE.Vector3(0,-100,0), geometry:boxgeometry, material:cupcakematerial}),
      ReactTHREE.Mesh({position:new THREE.Vector3(0, 100,0), geometry:boxgeometry, material:creammaterial})
    );
  }
});
